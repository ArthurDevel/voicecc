Twilio Voice Setup

1Create a Twilio account
Sign up at twilio.com/try-twilio. The free trial includes a phone number and enough credits for testing.
2Get a phone number
In the Twilio console, go to Phone Numbers → Buy a Number. Pick any number with voice capability.
3Copy your Account SID and Auth Token
Find both on the Twilio console dashboard under Account Info.
4Install ngrok
ngrok exposes your local server to the internet so Twilio can reach it. It will be started automatically when you launch the Twilio server.
Install from ngrok.com/download or via: brew install ngrok
